## R3E Shared Memory Limitations
The R3E API is more limited than AC EVO - it provides:
- Total steering force (`Player.SteeringForce`)
- Tire data (pressure, temp, wear, grip, load, speed)
- Suspension deflection
- Car state (speed, rpm, throttle, brake, gear, steering input)
- But NOT individual wheel forces or slip values
