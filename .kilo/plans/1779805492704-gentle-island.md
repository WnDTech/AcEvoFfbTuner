# Plan: Interactive Setup Wizard Overlay

## Overview

Create a step-by-step guided setup wizard overlay that helps new users configure their wheelbase FFB effects interactively while their game is running. The wizard is a draggable/resizable transparent overlay (same pattern as existing `CalibrationWizardOverlay`) that walks users through tuning the most impactful sliders, provides safety warnings for strong wheelbases, and saves the result with a user-chosen name and scope (per-car / per-track / per-game / general).

## Flow Summary

1. User opens app → HomePage shows prominent "⚡ SETUP WIZARD" button
2. User clicks → `SetupWizardOverlay` opens (topmost, transparent, draggable)
3. Step 0: Safety briefing, prerequisites check, device status
4. Steps 1–5: Guided slider tuning with live telemetry feedback + "HOLD WHEEL" safety warnings
5. Step 6: Name + scope selection → Save
6. Profile saved, auto-loaded, overlay closes

## Safety Warning Logic

- If `WheelMaxTorqueNm >= 8`: Show red pulsing banner "⚠ DANGER: Strong wheelbase detected. HOLD THE WHEEL FIRMLY during all force adjustments. Keep hands and face clear."
- If `WheelMaxTorqueNm < 8`: Show amber info banner "Hold the wheel during adjustments"
- Every step that changes force output (Steps 1, 2, 3) re-displays the warning
- Step where user can release: "✓ Forces adjusted. You can release the wheel."

## Profile Scope Save Dialog (Step 6)

User enters a profile name (textbox, pre-filled with suggested name like "My Profile v1").
Below: 5 radio buttons for scope:
- ○ General (works for any game/car/track)
- ○ Per Game (auto-fills GameMatch = current game name)
- ○ Per Car (auto-fills CarMatch = detected car model)
- ○ Per Track (auto-fills TrackMatch = detected track name)
- ○ Per Car + Track (fills both)
