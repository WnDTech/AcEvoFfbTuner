## index.html Structure

### Sections (top to bottom):
1. **Navigation** — Fixed top bar with logo, nav links (Features, Equalizer, Hardware, Screenshots, Download), mobile hamburger, GitHub CTA button
2. **Hero** — Full-viewport hero with app tagline, description, download + GitHub buttons, stats (15 pipeline stages, 1kHz output, 333Hz telemetry, 10-band EQ), animated floating steering wheel visual
3. **Key Highlights** — 3-card grid: 15-Stage DSP Pipeline, 10-Band FFB Equalizer, 1 kHz Output Interpolation
4. **Pipeline Features** — Grid of 15 cards covering all pipeline stages with accent colors, stage numbers, SVG icons
5. **Equalizer** — Visual section with animated 10-bar EQ visualization + band labels
6. **Hardware Support** — Grid of vendor cards (Fanatec, Moza, Simucube, Logitech, Asetek, VNM, Others/DirectInput) with active/stub status
7. **Screenshots Gallery** — Grid of screenshot cards with placeholder images, lightbox modal for full-size view. Drop your PNGs into `assets/img/screenshots/`
8. **Download CTA** — Call-to-action with download button linking to GitHub releases, version info, system requirements table
9. **Footer** — Branding, nav links, copyright, GitHub link

### Technical Details
- Pure HTML/CSS/JS — no frameworks, no build step
- Self-hosted fonts via Google Fonts (Inter, JetBrains Mono)
- SVG icons inline (no icon library dependency)
- Responsive: mobile, tablet, desktop
- Links to: `https://github.com/WnDTech/AcEvoFfbTuner` and `https://github.com/WnDTech/AcEvoFfbTuner/releases/latest`
- All data pulled from project (version 1.9.0, feature list from AGENTS.md, color theme from App.xaml)

### Screenshot Instructions (for user)
Drop screenshot PNGs into `website/assets/img/screenshots/` and update the `<img>` src attributes in the screenshots section of index.html. Each card has a `screenshot-placeholder` div that shows an upload prompt until real images are added.
